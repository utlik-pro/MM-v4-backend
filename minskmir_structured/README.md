## Структурированная база знаний по ЖК «Минск Мир»

Этот раздел содержит единый, чистый и без дублирования источник информации о ЖК «Минск Мир» и бизнес-логике общения с клиентами. Материалы агрегированы из текущих файлов проекта и сведены в компактные разделы.

### Состав папки
- `01-overview.md` — обзор комплекса (ключевые факты, инфраструктура, управление, гарантии)
- `02-education.md` — образовательная инфраструктура (сады, школы, статусы)
- `03-parking-sport.md` — паркинги и спортивная инфраструктура
- `04-finance.md` — рассрочка и кредитные программы (без дублирования)
- `05-delivery-schedule.md` — сроки сдачи домов (консолидировано по кварталам)
- `06-price-navigation.md` — навигация по ценовым категориям и правила при подборе
- `07-communication-playbook.md` — бизнес‑логика диалогов, скрипты и валидации
- `08-quarters-index.md` — индекс кварталов и быстрые ссылки на первоисточники

### Принципы структуры
- Без дублирования: каждая тема раскрывается в одном файле; в других файлах — только ссылки.
- Источник истины: детали берутся из исходных файлов в `quarters/` и `elevenlabs_rag/`.
- Навигация: все перекрёстные ссылки внутри папки и на исходники — относительные.

### Как пользоваться
1) Начинайте с `06-price-navigation.md` при любых вопросах по бюджету/ценам.
2) Для уточнения характеристик района — `01-overview.md` и профильные файлы (образование, паркинги, спорт).
3) Для условий оплаты — `04-finance.md`.
4) Для сроков по домам — `05-delivery-schedule.md`.
5) Для клиентской коммуникации — `07-communication-playbook.md`.
6) Для перехода к деталям по кварталам — `08-quarters-index.md`.

Последнее обновление: автоматически при правках исходных документов в проекте.
