# Daily Report Command

## Command: `/daily-report`

## Description
Generates a comprehensive daily HR report with key metrics and insights.

## Usage
```
/daily-report [options]
```

## Parameters
- `--date`: Report date (optional, default: today)
- `--format`: Output format (pdf, html, excel) (optional, default: html)
- `--recipients`: Email recipients (optional)
- `--include-charts`: Include data visualizations (optional, default: true)
- `--sections`: Specific sections to include (optional, default: all)

## Examples
```
/daily-report
/daily-report --date 2024-01-15 --format pdf --recipients "hr@company.com"
/daily-report --sections "applications,interviews,hires"
```

## Workflow
1. Collect data from multiple sources
2. Calculate daily metrics and KPIs
3. Generate insights and trends
4. Create visualizations
5. Format report
6. Distribute to stakeholders
7. Store report for historical tracking

## Report Sections
- New applications received
- Interviews scheduled/completed
- Job offers made/accepted
- Messages processed
- LinkedIn post performance
- Key metrics summary
- Action items and follow-ups

## Output
- Formatted daily report
- Key metrics dashboard
- Trend analysis
- Recommendations
- Action items
- Historical comparison
